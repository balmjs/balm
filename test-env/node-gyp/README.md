# Solution

- `.npmrc`

  ```ini
  registry=https://registry.npm.taobao.org
  sass_binary_site=https://npm.taobao.org/mirrors/node-sass
  phantomjs_cdnurl=https://npm.taobao.org/mirrors/phantomjs
  ```

- Environment variables

  | Variable name    | value                                    |
  | ---------------- | ---------------------------------------- |
  | SASS_BINARY_SITE | https://npm.taobao.org/mirrors/node-sass |
  | PHANTOMJS_CDNURL | https://npm.taobao.org/mirrors/phantomjs |
